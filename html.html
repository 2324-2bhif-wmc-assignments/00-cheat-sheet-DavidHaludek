Table



                    
Here is the general structure of a table:

  Table Header 1	      Table Header 2	      Table Header 3
Table cell 1	        Table cell 2	            Table cell 3
Table footer cell 1     Table footer cell 2	    Table footer cell 3
Source code:


<table>
    <thead>
        <tr>
            <th>Table Header 1</th>
            <th>Table Header 2</th>
            <th>Table Header 3</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Table cell 1</td>
            <td>Table cell 2</td>
            <td>Table cell 3</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Table footer cell 1</td>
            <td>Table footer cell 2</td>
            <td>Table footer cell 3</td>
        </tr>
    </tfoot>
</table>
